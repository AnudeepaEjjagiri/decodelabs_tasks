Personal Portfolio Website

  Overview
This is a responsive Personal Portfolio Website built using HTML, CSS, and JavaScript.  
It is designed to showcase my profile, skills, projects, and contact information in a clean and professional layout.

This project was created as part of my internship learning journey to improve frontend development skills.



 Features
- Responsive and modern UI design
- Navigation bar with smooth scrolling
- Dark / Light mode toggle 
- Live digital clock 
- Skills, Projects, and Internship sections
- Contact form UI
- Mobile-friendly layout


 Tech Stack
- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (DOM manipulation)
